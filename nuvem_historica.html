Nuvem Histórica

11 screens, a giant infra-red touch frame, and a c++ application developed using Cinder; that’s what it took to build the historical cloud for one of the largest construction corporations in Brazil.

Each screen has a set of items that contains the company landmark history, photos, and other related items (everything in 3 languages, Portuguese, Spanish, and English). Touching the background affects a particle system that is rendered as a delauney mesh. A video can be seen here

This work was done inside Índice studio in partnership with Venosa Design.

The main designers for this project were Jonattas Poltronieri (Art direction) and Igor Sturion.
